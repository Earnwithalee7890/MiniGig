# Changelog
All notable changes to the **MiniGig** project will be documented in this file.
## [1.3.0] - 2026-04-10
### Added
- Centralized constants in `src/constants/index.ts`.
- Custom hooks for MiniPay and Constants management.
- Comprehensive NatSpec documentation to `MiniGig.sol`.
- Project Architecture section to `README.md`.

### Improved
- Refactored `App.tsx` by extracting components.
- Improved type safety for contract data rendering.
- Updated `.gitignore` with better tool support.

## [1.2.0] - 2026-04-03
### Added
- Dedicated `ROADMAP.md` and `CONTRIBUTORS.md`.
- Shared type definitions in `src/types/index.ts`.
- Utility CSS classes for text and layout.
- Social sharing meta tags to `index.html`.
- Branching strategy to `CONTRIBUTING.md`.

### Improved
- Increased TypeScript strictness.
- Enabled Solidity optimizer in Hardhat.
- Added detailed RPC examples to `.env.example`.
- Expanded `.gitignore` rule set.
- Detailed comments in `MiniGig.sol` and `main.tsx`.
- Updated LICENSE and package metadata.

## [1.0.0] - 2026-04-01
### Added
- Initial deployment of MiniGig contracts to Celo.
- Basic React + Vite frontend with MiniPay integration.
- Implicit wallet connection.
- Glassmorphism UI components.
