# MiniGig Roadmap 🗺️

This document outlines the planned work for **MiniGig** as we grow in the Celo MiniPay ecosystem.

## ✅ Phase 1: Foundation (Completed)
- [x] Initial smart contract deployment.
- [x] Implicit wallet connection for MiniPay.
- [x] Premium glassmorphism UI/UX.
- [x] Daily check-in system on-chain (Mainnet).

## 🚀 Phase 2: Engagement (Q2 2026)
- [ ] **Leaderboard v1**: Display top users by streak and rewards.
- [ ] **Task Verification**: Automated social media task verification.
- [ ] **MiniPay Payouts**: Optimized gas abstraction for reward claims.

## 📈 Phase 3: Expansion (Q3 2026)
- [ ] **MiniGig SDK**: For developers to integrate task listings.
- [ ] **Mobile Notifications**: For task reminders.
- [ ] **Multi-Currency Support**: Earn rewards in cUSD, cEUR, and CELO.

## 🌐 Phase 4: Decentralization (Q4 2026)
- [ ] **DAO Governance**: Community voting on new task categories.
- [ ] **Decentralized Reputation**: Talent Protocol integration for user XP.
