# Security Policy

## Supported Versions

MiniGig follows the standard security support for our versions. Currently only the `main` branch is actively supported.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.0  | :white_check_mark: |
| < v1.0.0| :x:                |

## Reporting a Vulnerability

We take the security of MiniGig seriously. If you have discovered a vulnerability, please report it privately.

**Please do not open a public issue.**

To report a vulnerability, please contact us at security@minigig.celo with a detailed description of the issue. We will respond within 48 hours.

What to include:
- A clear description of the vulnerability.
- Step-by-step instructions to reproduce.
- Any potentially affected components.

We appreciate your effort to keep MiniGig safe for all users on the Celo network.
